Custom WordPress plugin that adds **secure warranty validation**, **autofill**, and **language toggle** for **Contact Form 7**.

## Features
- CF7 server-side validation (PHP)

- Language toggle EN ‚Üî AR
- Alerts for invalid warranties

## Installation
1. Upload the folder to WordPress:
wp-content/plugins/warranty-system-plugin/

2. Activate via WordPress Plugins menu
3. Ensure Contact Form 7 is installed
4. Add a text field named `serial-number` to your form


## Customization
- Add warranty numbers in `warranty-system.php` or `test-warranty.js`
- Adjust validation periods
- Customize CSS/JS for design
